# Hello World Add-on

This add-on is a minimal scaffold for future energy optimization work in the energyHome project.

## Usage

1. Install the add-on from the energyHome add-on repository.
2. Start the add-on.
3. Open the **Log** tab to confirm `Hello World` appears every 30 seconds.

## Notes

- The add-on runs as a background service (`startup: application`).
- No configuration is required.
